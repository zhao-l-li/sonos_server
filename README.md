# Sonos Server

Getting Started
---------------
1. install docker
2. install docker-compose
3. clone repository: `git clone --recursive git@github.com:zhao-li/sonos_server.git`
4. run bootstrap.sh: `./bootstrap.sh`
5. start server: `docker-compose up`
6. start playlist: `curl http://localhost:5005/master%20bedroom/favorite/playlist`
